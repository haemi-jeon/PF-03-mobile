# 웰체크 다이어트

대웅제약의 웰체크다이어트 서비스를 이용하는 사용자 및 코치에게 제공되는 하이브리드 모바일 앱을 위한 모바일 Web 화면입니다.

---

<a href="https://haemi-jeon.github.io/PF-03-mobile/" target="_blank">DEMO 이동</a>

---

### 화면 구조

- 앱초기 / 회원가입 / 로그인
- 홈
  - 캘린더
  - 필수 체크
  - 혈당 리포트 (구독정보)
  - 단식 타이머
  - 오늘의 콘텐츠
  - 오늘의 미션
  - 오늘의 기록
- 알림
  - 공지사항
  - 알림내역
- 내정보
  - 나의증상
  - 구독정보
  - 콘텐츠
  - 웰다 스토어
  - FAQ
  - 알림설정
  - 고객센터
  - 이용약관
- 입력
  - 단식
  - 신체기록
  - 식사
  - 운동
- 기록
  - 신체기록
  - 혈당기록
- 쳇
- 커뮤니티
